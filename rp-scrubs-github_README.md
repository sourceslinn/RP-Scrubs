# RP Scrubs Web & Database

Sitio web minimalista, vintage y caricaturesco diseñado para **RP Scrubs** (marca de ropa quirúrgica). 

## 🚀 Tecnologías utilizadas
* **HTML5**
* **CSS3** (Animación integrada y efectos Gaussian Blur)
* **JavaScript (Vanilla)**
* **SQL** (Estructura relacional e inventario inicial)

## 📁 Estructura del repositorio
* `index.html` - Estructura de la página y vinculación de componentes.
* `style.css` - Estilos globales, paleta de colores oficial, tipografías y animaciones.
* `app.js` - Gestión dinámica del carrito de compras y lógica interactiva de componentes (FAQ).
* `database.sql` - Script completo para montar la base de datos relacional y variaciones de inventario.
